# Nova WordPress Theme

A clean and simple WordPress block theme, built for speed and simplicity.

## Description

Nova is a modern WordPress block theme that embraces Full Site Editing (FSE). Built with simplicity in mind, it provides a solid foundation for building fast, accessible websites.

## Features

- Full Site Editing (FSE) support
- Lexend font family included
- Responsive typography with fluid font sizes
- Custom color palette
- Clean, semantic HTML
- Accessibility-ready
- Translation-ready

## Requirements

- WordPress 6.7 or higher
- PHP 8.4 or higher

## Installation

1. Download the theme
2. Upload to /wp-content/themes/
3. Activate in WordPress admin

## File Structure

```
nova/
├── assets/           # Theme assets (CSS, fonts, images)
├── inc/              # PHP classes and functions
│   ├── classes/      # PHP classes
│   ├── config/       # Configuration files
│   └── functions/    # Helper functions
├── parts/            # Template parts
├── patterns/         # Block patterns
├── templates/        # Page templates
├── functions.php     # Theme bootstrap
├── style.css         # Theme header
└── theme.json        # Theme configuration
```
