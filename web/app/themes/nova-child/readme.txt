Child theme for the Nova WordPress theme.

== Description ==

Nova Child is a child theme for the Nova WordPress theme. It allows you to customize the Nova theme without modifying the parent theme files, ensuring your customizations remain intact after theme updates.

== Features ==

* Maintains all functionality of the parent Nova theme
* Easy to customize to meet your specific needs
* Clean, simple structure

== Installation ==

1. Make sure you have the parent Nova theme installed.
2. Upload the nova-child theme folder to your /wp-content/themes/ directory.
3. Activate the Nova Child theme from your WordPress admin dashboard.
